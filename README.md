# CrestSow

**CrestSow** is a tech education platform providing CRT (Campus Recruitment Training), projects, internships, workshops, hackathons, placement guidance, resume building, and company-specific trainings.

## Services

- Campus Recruitment Training (CRT)
- Hands-on Projects
- Internships
- Workshops & Hackathons
- Placement Guidance
- Resume Building
- Trending Courses
- Company-Specific Trainings

## Training Domains

AIML, AI, Embedded Systems, IoT, Python, C, Java, DSA (Python / C / Java), Python Full Stack, Java Full Stack, MERN Stack, Aptitude & Reasoning, Soft Skills, Verbal.

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- [Tailwind CSS](https://tailwindcss.com/) v4
- [Framer Motion](https://www.framer.com/motion/)
- [Embla Carousel](https://www.embla-carousel.com/)
- [Lucide Icons](https://lucide.dev/)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
